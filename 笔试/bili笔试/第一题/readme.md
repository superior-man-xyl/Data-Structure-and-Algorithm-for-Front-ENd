给定一个正整数数组，寻找数组当中重复次数第N多(N∈(1, n))的数字和对应的重复次数。若重复次数一致，则数字较小的排序在前；若N没有对应的数字，则输出-1

输入第一行为一个数组

输入第二行为需要查询的第几大的编号

举例：

=========== 输入

[1,3,2,3,2,2,1,2]

1 输出

2 4

=========== 输入

[1,3,2,3,2,2,1,2]

2 输出

1 2

=========== 输入

[1,3,2,3,2,2,1,2]

4 输出

-1 -1

===========


输入描述
输入第一行为一个数组（不超过500）

输入第二行为需要查询的第几大的编号

输出描述
输出第一多的数字，以及出现次数，空格隔开


样例输入
[1,3,2,3,2,2,1,2]
1
样例输出
2 4
